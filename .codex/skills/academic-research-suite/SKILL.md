# Academic Research Suite

Adapted from Codex academic research skill collections and co-researcher style workflows.

## Use After Topic Scoping

This is not the first skill for raw search. Use it when a research cluster, paper queue, method family, or technical question already exists.

## Procedure

1. Define the research cluster, method family, or technical question.
2. Build a small, high-quality bibliography.
3. Synthesize methods, datasets, questions, and gaps.
4. Map prerequisites, implementation choices, and experiments to the cluster.
5. Produce positioning notes for future reading, derivations, experiments, or project writeups.

## Output

- Cluster summary.
- Key papers table.
- Method comparison.
- Technical positioning.
- Missing reading list.
